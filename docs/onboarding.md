# User Onboarding Procedure

## Purpose
To ensure secure, consistent, and efficient onboarding of new staff into IT systems.

## Steps
1. HR submits onboarding request
2. Manager approval
3. Account creation
4. Access provisioning
5. Equipment allocation
6. User orientation
7. Documentation update

## Systems Involved
- Active Directory
- Office 365
- SharePoint
- Service Desk System
- Asset Register

## Security Controls
- Role-based access control
- MFA enforcement
- Password policies
- Audit logging
